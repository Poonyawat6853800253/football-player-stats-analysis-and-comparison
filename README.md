# Football Players Stats 2024/2025 Analysis ⚽📊

This project analyzes football player statistics from the 2024/2025 season using Python.  
It includes exploratory data analysis (EDA), player performance comparison, xG-based evaluation, and a simple player recruitment scoring system for transfer analysis.

---

## 📌 Project Objectives

The main objectives of this project are:

- To explore football player performance data from the 2024/2025 season
- To identify top goal scorers and assist providers
- To compare actual goals with expected goals (xG)
- To analyze players by league and position
- To identify young talents with strong performance
- To create per-90-minute metrics for fair player comparison
- To build a simple recruitment score for forward players

---

## 📂 Dataset

The dataset contains football player statistics from the 2024/2025 season.
